Weather App Project

Project Overview

A weather app that allows users to enter a specific location and automatically detects their location. The app displays high and low temperatures and includes input validation for user-entered data.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • API Integration

Key Features

  • User input for location
  • Automatic location detection
  • Display of high and low temperatures
  • Input validation for user-entered data

Challenges

Overcoming challenges during development, such as handling API responses and ensuring a smooth user experience with input validation.
